# Job Blaster (MVP)

Personal job tracker: capture jobs fast, Kanban tracking, short tailored cover
notes, and never miss follow-ups.

## Features

- **Job Capture**: Quick job saving with URL parsing
- **Kanban Board**: Visual job status tracking (Saved → Applied → Interview →
  Offer → Rejected)
- **Application Tracking**: Resume management and application notes
- **Cover Letter Templates**: Reusable templates with smart placeholders
- **Follow-up Reminders**: Deadline alerts and automated notifications
- **CSV Export**: Data portability and backup

## Tech

React + Vite + Tailwind + TanStack Query • Node + Express + Prisma • Postgres
(Supabase/Neon)

- **/docs/MVP-plan.md** - Full MVP specification
- **/docs/TEMPLATES.md** - Cover letter templates guide
